JsRestBin API


The JsRestBin API allows third-party application developers to operate the application via REST calls.

Sample call

http:// /api/bins/cq5mi/requests/latest

REST Methods

Call Description
GET Retrieves information.
PUT Updates existing information.
POST Creates new information.
DELETE Removes existing information.

API Specification

Endpoint Description
/bins Provides a list of all bins
/:bin_id Shows a particular bin
/requests Provides a list of all recorded requests for the bin
/:request_id Shows a particular request of the given bin
/latest Shows the last recorded request
